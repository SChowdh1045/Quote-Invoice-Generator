# Quotes and Invoices Generator

This Java application is designed for our family business, enabling the creation of personalized quotes and invoices for our clients. Utilizing the iTextPDF library, it facilitates the seamless generation of PDF documents that incorporate our company's branding elements.

## Overview

The core functionality of this application includes:

- Generating customized quotes and invoices as PDF files.
- Incorporating company branding, such as logos and color schemes, into the documents.
- Providing a simple and intuitive interface tailored for our family business's specific needs.

This project is a personal endeavor, created to streamline our business operations and ensure a professional presentation to our clients.

## Technology

- **Language**: Java
- **Library**: iTextPDF for PDF generation
